# Code of Conduct

## Our Standard

Dockistrate should be a practical, respectful project space for operators,
contributors, and maintainers. Participants are expected to communicate clearly,
assume good faith, and focus discussion on the technical work.

Unacceptable behavior includes harassment, personal attacks, sustained
disruption, publishing private information without permission, and any conduct
that would make participation unsafe or hostile.

## Scope

This code applies in project spaces, including issues, pull requests,
discussions, release channels, and maintainer-directed community forums.

## Enforcement

Report conduct concerns privately to `@nourae` on GitHub.

Maintainers may remove comments, close threads, restrict participation, or take
other proportionate action when behavior violates this code. Enforcement should
be consistent, documented where appropriate, and focused on restoring a
productive project environment.

## Reporting

Report conduct concerns privately to the enforcement contact. Include links,
screenshots, or context where possible, and avoid sharing sensitive personal
information beyond what is necessary to investigate the concern.
